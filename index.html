About

To get started: Just copy the whole gitRepo and start replacing the content of this index.html file.

Wrap all content inside the article element with id="all-content".

Wrap all your chapters or sections content inside a section element each.
Use p element for every regular text paragraph.
The br element is used to force line breaks.
This about section could be replaced with an abstract or a summary. Take care to label the elements heading h2 id tho something meaningful.

Start to change the tufte-customized.css CSS-document to change the style according to your needs.

All dark mode styling goes into one section at the beginning, style-changes for small screens (mobile view) are place in the second section and all style-changes that are only applied when printing the document is compiled at the end of the css-document.

If you want to print your document with a proper layout, uncomment the corresponding section inside the head of this document to active pagedJS. Then the browser shows the print-preview (all the colorful markers are only visible on screen). Please have a look at this great project's website for further details on printing and formatting with html/CSS for print. Don't forget to place the div element for the print-version of the table of content to place where it should appear.

If you want to put in an equation or other math, use MathML format and wrap the content of the math element into a p element with class="math-block":

x = − b ± b2 − 4ac 2a

(There are a lot of useful MathML editors and converters around…)
